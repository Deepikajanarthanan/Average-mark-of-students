# Student Average Analyzer 📊

This project calculates the average marks of students using Python.

## 🚀 Features
- Calculate average marks
- Supports multiple subjects
- Simple and easy to use

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Open terminal
2. Run:
   python main.py
